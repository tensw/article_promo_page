# Research Story Curation Page Specification

> 논문 1편에 대한 큐레이션 페이지 구성 명세
> 대상: 성균관대학교 학술정보관
> 시리즈: Research Story 2026년 1호

---

## 개요

논문 한 편을 다양한 포맷(영상, 매거진, 카드뉴스)으로 가공하여
하나의 페이지에서 통합 제공하는 큐레이션 랜딩 페이지.

---

## 페이지 구조

### 1. Header

| 항목 | 설명 |
|------|------|
| 위치 | 페이지 최상단, 고정(sticky) |
| 좌측 | 성균관대학교 로고 (이미지) |
| 우측 | 시리즈 타이틀: `Research Story: 2026년 1호` |
| 스타일 | 깔끔한 화이트/라이트 배경, 하단 경계선 |
| 반응형 | 모바일에서 로고 축소, 타이틀 줄바꿈 허용 |

---

### 2. Banner (아티클 카드)

2단 레이아웃. 논문의 첫인상을 결정하는 핵심 영역.

#### 2-1. 좌측: 카드뉴스 스타일 헤드라인

| 항목 | 설명 |
|------|------|
| 헤드라인 | 논문 내용을 자극적·대중적으로 재구성한 타이틀 (1~2줄) |
| 서브텍스트 | 핵심 내용 요약 2~3줄 |
| 톤앤매너 | 카드뉴스 스타일 — 짧고 임팩트 있는 문장, 숫자 강조 |
| 배경 | 그라데이션 또는 논문 관련 이미지 오버레이 |
| 예시 헤드라인 | "서울 집값 18% 폭등, 지방은 -26% 폭락<br>한국 주택시장에 무슨 일이?" |
| 예시 서브텍스트 | "한국은행이 경고하는 3가지 구조적 위험 신호.<br>전세 시대의 종말, 월세 60% 시대가 온다." |

#### 2-2. 우측: 숏폼 영상 (폰 프레임)

| 항목 | 설명 |
|------|------|
| 형태 | 스마트폰 모양 프레임 (9:16 비율) |
| 콘텐츠 | 논문 소개 숏폼 영상 (60초 내외) |
| 소스 | YouTube Shorts / Instagram Reels 임베딩 또는 자체 호스팅 MP4 |
| 프레임 디자인 | 둥근 모서리, 노치/다이나믹 아일랜드 표현, 그림자 효과 |
| 인터랙션 | 클릭 시 재생, 하단에 "숏폼으로 1분 만에 이해하기" 캡션 |

#### Banner 레이아웃

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│   ┌──────────────────────┐   ┌───────────────────┐      │
│   │                      │   │   ┌───────────┐   │      │
│   │   서울 집값 18% 폭등,  │   │   │           │   │      │
│   │   지방은 -26% 폭락     │   │   │  숏폼 영상  │   │      │
│   │                      │   │   │  (9:16)    │   │      │
│   │   한국은행이 경고하는    │   │   │           │   │      │
│   │   3가지 구조적 위험...  │   │   │           │   │      │
│   │                      │   │   └───────────┘   │      │

```

### 3. Long-form Video (롱폼 소개 영상)

| 항목 | 설명 |
|------|------|
| 위치 | 배너 하단, 전체 너비 |
| 형태 | YouTube 임베딩 (16:9 비율) |
| 콘텐츠 | 논문 심층 해설 영상 (5~15분) |
| 섹션 타이틀 | "Deep Dive: 논문 심층 해설" 또는 유사 표현 |
| 반응형 | `padding-bottom: 56.25%` 기반 비율 유지 |
| 추가 요소 | 영상 하단에 논문 원문 메타정보 (제목, 저자, 출처, 연도) 표시 |

#### 레이아웃

```
┌─────────────────────────────────────────────────────────┐
│  DEEP DIVE                                              │
│  ┌─────────────────────────────────────────────────┐    │
│  │                                                 │    │
│  │            YouTube 임베딩 (16:9)                  │    │
│  │                                                 │    │
│  └─────────────────────────────────────────────────┘    │
│  논문 제목 · 저자 · 출처 · 연도                           │
└─────────────────────────────────────────────────────────┘
```

### 4. Document Viewer (매거진 뷰어)

| 항목 | 설명 |
|------|------|
| 위치 | 롱폼 영상 하단 |
| 형태 | 매거진(HTML) 임베딩 뷰어 |
| 콘텐츠 | `magazine/` 디렉터리의 논문 매거진 HTML |
| 섹션 타이틀 | "Research Magazine: AI 분석 리포트" |
| 임베딩 방식 | `<iframe>` 으로 매거진 HTML 로드 |
| 뷰어 기능 | 페이지 네비게이션 (1/3, 2/3, 3/3), 전체화면 버튼 |
| PDF 다운로드 | "PDF 다운로드" 버튼 (브라우저 print 기능 연동 또는 사전 생성 PDF 링크) |

#### 레이아웃

```
┌─────────────────────────────────────────────────────────┐
│  RESEARCH MAGAZINE                                      │
│  ┌─────────────────────────────────────────────────┐    │
│  │                                                 │    │
│  │         매거진 HTML iframe                        │    │
│  │         (A4 비율 뷰어)                            │    │
│  │                                                 │    │
│  └─────────────────────────────────────────────────┘    │
│  ◀ Page 1/3 ▶                    [ PDF 다운로드 ]       │
└─────────────────────────────────────────────────────────┘
```

## 데이터 입력 구조

페이지 생성 시 필요한 입력값:

```yaml
series: "Research Story: 2026년 1호"
paper:
  title: "Characteristics of Recent Housing Market and Impact on Financial System"
  title_ko: "최근 주택시장 특징 및 금융시스템 영향 분석"
  authors: "Kim Dongwook, Kim Woo-seob"
  source: "Bank of Korea, Financial Stability Report"
  year: 2025
  month: 12

banner:
  headline: "서울 집값 18% 폭등, 지방은 -26% 폭락"
  subheadline: "한국 주택시장에 무슨 일이?"
  description: |
    한국은행이 경고하는 3가지 구조적 위험 신호.
    전세 시대의 종말, 월세 60% 시대가 온다.
  short_video_url: "https://youtube.com/shorts/XXXXX"

longform:
  youtube_url: "https://youtube.com/watch?v=XXXXX"

magazine:
  html_path: "../magazine/housing-market-magazine.html"
  pdf_path: "../magazine/housing-market-magazine.pdf"
  total_pages: 3
```

## 기술 스택

| 항목 | 선택 |
|------|------|
| 구현 방식 | 단일 HTML 파일 (self-contained) |
| 스타일링 | Inline CSS (외부 의존성 없음) |
| 영상 임베딩 | YouTube iframe API |
| 매거진 뷰어 | iframe + JS 페이지 네비게이션 |
| 반응형 | CSS media queries (모바일 대응) |
| 폰트 | 시스템 폰트 + Google Fonts (Noto Sans KR) |

## 디자인 톤앤매너

| 요소 | 방향 |
|------|------|
| 전체 톤 | 학술적이면서도 대중 친화적 — "학술 매거진" 느낌 |
| 색상 | 성균관대 CI 기반 (그린 계열) + 뉴트럴 |
| 타이포그래피 | 헤드라인은 볼드·대형, 본문은 가독성 우선 |
| 배너 | 시선을 끄는 카드뉴스 스타일, 숫자·키워드 강조 |
| 영상 영역 | 충분한 여백, 깔끔한 프레임 |
| 매거진 영역 | A4 문서 느낌의 뷰어, 미니멀 컨트롤 |
